Requirements
============

* Redmine version >= 2.6.0
* Ruby >= 2.0.0
* Gem package: see PluginGemfile


Installation
============

Install redmine_tweaks

.. code-block:: bash

	$ cd $REDMINE_ROOT
  $ git clone git://github.com/alexandermeindl/redmine_tweaks.git plugins/redmine_tweaks
  $ bundle install --without development test
